Conclusiones
------------
->	Con el comando "npm install express --save", Instalamos la librería Express de modo local.
	Es decir, esta librería estará disponible solo para el proyecto donde se ejecutó el código.

-->	Instalamos la herramienta Generator de Express con el comando "express-generator". Esta
	herramienta nos permite integrar a nuestro proyecto el esqueleto estandar recomendado para
	utilizar Express. Sin embargo, existen muchas otras formas de estructurarlo.
